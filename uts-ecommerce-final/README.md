**Deskripsi:**
Project ini adalah website eCommerce sederhana berbasis PHP untuk memenuhi tugas UTS mata kuliah eCommerce.  
Aplikasi ini menampilkan daftar produk, memungkinkan pengguna menambahkan ke keranjang, dan melakukan checkout sederhana.

**Fitur:**
- Menampilkan daftar produk
- Menambahkan produk ke keranjang
- Menampilkan total belanja
- Checkout sederhana
